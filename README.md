Set of tools for analysis/postproduction of Quantum Espresso runs

Tools:
- qe_analyze.py: analyze pw.out for convergence and errors.
- qe_pw_to_poscar.py: extract the last geometry from pw.out and write a POSCAR.
- qe_pwout_to_pwin.py: extract the last geometry from pw.out and write a pw.in file (QE input required for file template).
